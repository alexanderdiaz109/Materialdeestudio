💡 ¿Cómo estudiar aquí?

Este manual está diseñado con ejemplos bilingües. El texto en verde es la estructura en inglés, seguido de su traducción al español para que tu cerebro haga la conexión rápida.

Usa el menú lateral para navegar por los 4 módulos principales.

📦 Módulo 1: Fundamentos

Sujetos, Posesiones, Artículos y Cantidades.

👥 1.1 Pronombres y Posesivos

Tipo Ejemplo Inglés ↔️ Español
Subject (Sujeto)
Hace la acción		 I code daily.Yo programo a diario.
Object (Objeto)
Recibe la acción		 The client pays me.El cliente me paga.
Possessive Adjective
Va antes de la cosa		 Where is my laptop?¿Dónde está mi laptop?
Possessive Noun ('s)
Añade 's al dueño		 Alexander's code is clean.El código de Alexander es limpio.

🎯 1.2 Demostrativos y Artículos

Distancia Demostrativo (This/That) Artículos (A/An)
Cerca This pixel art is great.Este pixel art es genial.These files are new.Estos archivos son nuevos. A (Sonido Consonante)
A database. / A project.
Lejos That server is offline.Ese servidor está desconectado.Those bugs are old.Esos bugs son viejos. An (Sonido Vocal)
An API. / An error.

📊 1.3 Plurales y Cuantificadores

Contables: Se cuentan (apps, servers). Incontables: No se cuentan (software, information).

Cuantificador Uso Key Ejemplo Bilingüe
Some Afirmativas (+) I have some new designs.Tengo algunos diseños nuevos.
Any Negativas (-) / Preguntas (?) Do you have any bugs?¿Tienes algún bug?
A lot of Grandes cantidades We receive a lot of updates.Recibimos muchas actualizaciones.

⚡ Módulo 2: Presente y Futuro

Estado actual, rutinas continuas y planes de desarrollo.

📌 2.1 To Be & There is/are (Ser, Estar, Existir)

Estructura Ejemplo Bilingüe
Verb To Be I am an IT student. The database is ready.Soy estudiante de TI. La base de datos está lista.
There is (Hay 1) There is a bug in the script.Hay un error en el script.
There are (Hay +2) There are five people in the team.Hay cinco personas en el equipo.

🔄 2.2 Simple Present & Rutinas

Para rutinas. Recuerda agregar -s o -es si el sujeto es He/She/It.

Frecuencia Ejemplo Bilingüe
Always / Often
(Siempre / A menudo)		 The script runs automatically. We always test the app.El script corre automáticamente. Siempre probamos la app.
Sometimes / Never
(A veces / Nunca)		 I sometimes study integral calculus.A veces estudio cálculo integral.

🚀 2.3 Progreso Ahora y El Futuro

Tiempo Verbal Situación Ejemplo Bilingüe
Present Continuous
(To be + ING)		 Acción sucediendo AHORA. We are developing the RuralDrive app right now.Estamos desarrollando la app RuralDrive ahora mismo.
Be going to Plan con intención sólida. I am going to build a news API.Voy a construir una API de noticias.
Will Decisión rápida / Promesa. Don't worry, I will fix the server.No te preocupes, yo arreglaré el servidor.

🕰️ Módulo 3: Pasado y Experiencias

Historias de código, proyectos terminados y experiencias.

⏪ 3.1 Past Simple & Pasado Continuo

Concepto Ejemplo Bilingüe
Past Simple
(Terminado: +ed o Irregular)		 I wrote the code and saved the file.Escribí el código y guardé el archivo.
Used to
(Hábitos pasados que ya no haces)		 I used to play games, now I design them.Solía jugar juegos, ahora los diseño.
Past Continuous
(was/were + ING)		 I was designing a 2D medieval game when the laptop crashed.Estaba diseñando un juego 2D medieval cuando la laptop se colgó.

🏆 3.2 Present Perfect (Experiencias de Vida)

Usa Have/Has + Participio Pasado para experiencias donde el tiempo exacto no importa, o acciones que siguen hasta hoy.

Palabra Clave Ejemplo Bilingüe
Since / For
(Desde / Por)		 We have worked with Diaz and Jarodn for months.Hemos trabajado con Diaz y Jarodn por meses.
Already / Just
(Ya / Acabo de)		 I have already finished. I have just compiled it.Ya he terminado. Acabo de compilarlo.
Yet
(Aún no / ¿Ya?)		 I haven't pushed the commit yet.Aún no he subido el commit.
Ever / Never Have you ever failed? I have never lost data.¿Alguna vez has fallado? Nunca he perdido datos.

🛠️ Módulo 4: Funciones y Comparaciones

Verbos modales para dar opiniones, y comparativos.

🚦 4.1 Modal Verbs

Regla de oro: No cambian (no llevan 's') y el siguiente verbo va en su forma base (sin 'to').

Modal Función Ejemplo Bilingüe
Can / Could Habilidad I can create pixel art.Puedo crear arte pixelado.
Should Consejo You should document your code.Deberías documentar tu código.
Must Obligación Fuerte We must deliver the Mandadito app by Friday.Debemos entregar la app Mandadito el viernes.
Might / May Posibilidad The client might ask for UI changes.El cliente podría pedir cambios de interfaz.

⚖️ 4.2 Comparativos y Superlativos

Tipo Regla Visual Ejemplo Bilingüe
Comparatives
(A vs B)		 Adjetivo corto: +er than
Adjetivo largo: more + adj		 The Lenovo LOQ is faster than my old PC.La Lenovo LOQ es más rápida que mi PC vieja.
Superlatives
(El Top 1)		 Adjetivo corto: the +est
Adjetivo largo: the most + adj		 This is the most efficient algorithm.Este es el algoritmo más eficiente.
