Navega por el menú lateral para acceder a los 23 temas gramaticales que seleccionaste, organizados de forma lógica. Los ejemplos están adaptados a tu entorno tecnológico y de desarrollo de software para que te sea más fácil asimilarlos.

📘 Módulo 1: Fundamentos

Pronombres, artículos, sustantivos y cuantificadores.

1.1 Subject/Object Pronouns & Possessives

Subject (Sujeto): I code the app. They test the software.
Object (Objeto): The client pays me. I send the file to them.
Possessive Adjective: This is my Lenovo laptop.
Possessive Noun ('s): It is Alexander's project.

1.2 Demonstratives & Articles

Cerca: This pixel art is cool (este) / These designs are new (estos).
Lejos: That server is down (ese) / Those bugs are annoying (esos).
A/An: A database (sonido consonante) / An API (sonido vocal).

1.3 Plurals, Countables & Quantifiers

Plurals: Generalmente +s (apps). Irregulares: man/men, data (es plural de datum).
Contables: servers, computers. Usan many (muchos), a few (unos pocos).
Incontables: software, information, time. Usan much (mucho), a little (un poco).
Some/Any: I have some ideas (+) / Do you have any questions? (?)

🟢 Módulo 2: Presente y Futuro

To be, rutinas, acciones continuas y planes.

2.1 Verb To Be & There is/are

To Be (Ser/Estar): I am an IT student. The database is ready. We are developers.
There is (Hay singular): There is a bug in the code.
There are (Hay plural): There are five people in the team: Diaz, Jarodn, Flota, Villanueva, and Cordova.

2.2 Simple Present & Frequency Adverbs

Simple Present: The script runs every morning. I code in Python. (Agrega -s/-es con He/She/It).
Frequency Adverbs: Van antes del verbo principal. We always test the app. I sometimes study integral calculus.

2.3 Present Continuous & Activities with ING

Present Continuous: Acciones que ocurren ahora (To be + verbo-ing). We are developing the RuralDrive app right now.
Activities: Cuando la actividad es el sujeto de la oración. Programming is fascinating. I enjoy designing video games.

2.4 Future Forms

Will (Decisión rápida / Predicción): Don't worry, I will fix the server tomorrow.
Be going to (Plan con intención): I am going to build a news API for Mexico.
Present Continuous (Plan fijo agendado): We are launching Mandadito next month.

🟠 Módulo 3: Pasado y Experiencias

Pasado simple, continuo y presente perfecto.

3.1 Past Simple & Used to

Pasado Simple (Acción terminada): We worked hard yesterday (Regular +ed). I wrote the documentation (Irregular).
Used to (Hábitos pasados): I used to play old games, now I develop Defensa del Muro.

3.2 Past Continuous & VS Past Simple

While (Acciones paralelas): I was coding while Diaz was testing.
When (Interrupción): I was designing a 2D game when the computer crashed.

3.3 Present Perfect & Time Expressions

Estructura: Have/Has + Participio pasado. Indica experiencia.
Since / For: I have studied IT since 2023. We have worked for three hours.
Already / Just: I have already finished. I have just compiled the code.
Yet: (Negativas/Preguntas) I haven't pushed the changes yet.
How long / Ever / Never: How long have you been a programmer? Have you ever failed a test? I have never lost data.

3.4 Present Perfect VS Past Simple

Past Simple (Sabes CUÁNDO): I finished the Homún local guides project in January.
Present Perfect (Importa la EXPERIENCIA): I have developed several mobile apps.

🟣 Módulo 4: Funciones y Comparaciones

Verbos modales, comparativos y superlativos.

4.1 Modal Verbs

Can / Could (Habilidad): I can speak English. I couldn't sleep yesterday.
Should (Consejo): You should document your code.
Must (Obligación): We must deliver the project by Friday.
Might / May (Posibilidad): The client might ask for changes.
Would (Condicional): I would like to learn machine learning and AI.

4.2 Comparatives and Superlatives

Comparatives (A vs B):
Cortos (+er): This laptop is faster than the old one.
Largos (more): This algorithm is more efficient than the previous one.

Superlatives (El Top 1 de un grupo):
Cortos (the +est): This is the fastest processor.
Largos (the most): It is the most important feature of the app.
